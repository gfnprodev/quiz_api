1. Настройка окружения
Вам нужно скопировать файл env.example и назвать его .env (точка в начале).
Откройте .env файл и укажите необходимые значения переменных окружения.

2. Запуск
В терминале введите: docker compose up ИЛИ docker-compose up

Docker Compose запустит проект. Приложение FastAPI будет доступно по адресу: http://localhost:8000
Для запроса: http://localhost:8000/questions/
JSON DATA:
{
    "questions_num": 0
}
Где 0 - число вопросов которое вы хотите получить

При успехе возвращает 200 status code.

Пример ответа:

{
  "id": 0,
  "question": "string",
  "answer": "string",
  "created_at": "2023-05-22T22:54:41.215Z"
}

Документация доступна по ссылке: http://localhost:8000/docs
